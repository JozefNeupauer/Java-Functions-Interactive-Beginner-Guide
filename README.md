# ☕ Java Functions — Interactive Beginner Guide

A single HTML file that teaches Java fundamentals through **live, interactive demos**.  
No setup. No install. Just open it and start playing.

## What's inside
- **27 cards** covering Strings, Arrays, Loops, Math, OOP, and Collections
- Every card has a **live demo** — type your own input and see the output update instantly
- Beginner-friendly explanations with real-world analogies
- Step-through loop visualizer (watch `i` change each iteration)
- Visual array & character boxes so you can *see* what the code is doing

## How to use
Just open `index.html` in any browser — no server needed.  

## Built with
Pure HTML, CSS, and vanilla JavaScript. Zero dependencies.
